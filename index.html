🎯 The Problem

Nigerians — especially women, professionals working late, and students — face daily safety friction. There's no fast, low-friction way to tell trusted contacts "I'm on my way home" or "I need help" without unlocking your phone, finding the right person, and typing. Existing options (WhatsApp share-location, calling a friend) are slow, social, and don't scale to emergencies.

Target user: Ada, 28, project manager in Lagos. Takes Bolt rides home after late meetings. Has two trusted contacts (sister + close friend).

🛠️ Tech Highlights

  • Single-file HTML app (~210 KB) — deploy to any static host
  • Real OpenStreetMap via Leaflet — embedded inline, zero CDN dependencies
  • Works on the worst Nigerian networks: library is self-hosted, tiles load progressively, error fallbacks for failed tiles
  • Real GPS via navigator.geolocation + on-screen permission flow
  • Native haptics for SOS (Android navigator.vibrate)
  • Shake detection via DeviceMotion API
  • No build step. No dependencies. No backend (yet — V2 will wire to Supabase like SafeSignal).

🏗️ Architectural Roadmap (V2 Migration)

To scale from a static prototype to a live infrastructure app without damaging network resilience metrics, V2 executes the following backend migrations:

1. Low-Overhead Supabase Authentication

Integrate passwordless OTP access via WhatsApp/SMS. If extreme regional congestion limits message transit, the architecture automatically drops to an offline "Guest Mode" powered by cached states in browser localStorage.

2. Real-Time Emergency Sync Layer

Initialize Supabase PostgreSQL tables to host active travel states. Live tracking events transfer via Supabase WebSockets (Realtime) to tracking portals. Under low-signal 2G operations, data retries cache locally and push incrementally via standard HTTP payloads.

3. Failover Trigger via Edge Functions

Deploy decoupled TypeScript functions to the Supabase Edge Network. The scripts execute immediately if an active trip ETA passes without a secure termination event, communicating with local SMS gateways (Termii/Twilio) to send distress telemetry via traditional telecom lines.
